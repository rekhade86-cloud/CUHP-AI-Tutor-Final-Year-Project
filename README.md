# CUHP-AI-Tutor-Final-Year-Project
CUHP AI Tutor is a locally hosted AI chatbot developed as a Final Year Project at Central University of Himachal Pradesh. It assists UG and PG students with academic learning through concept explanations and problem-solving, ensuring privacy, offline access, and low-cost deployment.

Step by Step Procedure
1. Download, install and setup ollama.exe from ollama wesite
2. Then run command on cmd ollama run phi:2.7b
3. Make a packages which I given to you under the common folder
4. Run cmd of common folder and run command py -m http.server 8000(do every time when use)
5. Open browser http://localhost:8000 and use your ai model(Do every time when use)

For error restart do again or contact me rekhade86@gmail.com
